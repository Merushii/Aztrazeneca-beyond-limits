# Tour Virtual de AstraZeneca

Este proyecto consiste en un tour virtual de las instalaciones de AstraZeneca, con un enfoque en la accesibilidad, interactividad y escalabilidad. Permite a los usuarios explorar las instalaciones de manera virtual utilizando imágenes equirectangulares para generar una vista 360, así como la presentación de modelos 3D en diversos puntos clave.

## Características principales

- Utiliza imágenes equirectangulares para generar una vista 360 de las instalaciones.
- Presenta modelos 3D en puntos específicos para una experiencia más inmersiva.
- Enfoque en accesibilidad, interactividad y escalabilidad para una experiencia de usuario mejorada.
- Compatible con diversos dispositivos, ya que se trata de una página web accesible desde cualquier navegador.

## Requisitos

No hay requisitos específicos para ejecutar este proyecto, ya que se trata de una página web que puede visualizarse en cualquier dispositivo con acceso a un navegador web.

## Versiones

- Framework de desarrollo web utilizado: Marzipano versión 0.10.2
- Plataforma de modelado 3D utilizada: Blender versión 4.0
- Exportacion de modelos 3D: Three.js versión 0.137.0

## Instalación y ejecución

Dado que el proyecto es una página web, no requiere instalación ni ejecución. Basta con acceder a la URL proporcionada para comenzar a disfrutar del tour virtual.
